# Travel Companion

## Tech Stack

- **Frontend:**

  - **Framework**: Next.js
  - **Language**: TypeScript
  - **Styling**: Tailwind CSS
  - **UI Components**: Radix UI
  - **Form Handling**: React Hook Form
  - **Data Validation**: Zod
  - **State Management**: React Context API

- **Backend:**

  - **Framework**: Next.js API routes
  - **Language**: TypeScript

- **Hosting:**
  - **Link**: Vercel

## How to Run the project?

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
